主程序在cofi.m中

网上数据的链接挂掉了，我用的是Andrew NG的数据

为了移植到其他语言会更方便，我几乎没有用向量化的计算，这导致程序在matlab下运行很慢（matlab下用for循环效率极低，因为这样不能利用并行处理，但在其他语言不会出现这种状况），
这可能导致如果用全部数据集我的程序会慢的不可接受，
所以我把数据集调小了，在主程序中可以调数据集的范围，详见注释。

程序会输出三张图片，第一张是数据集给出的用户评分的可视化，颜色不同象征数据大小不同，颜色相近象征数据大小相近，第一张图片有大量的蓝色表明这个值是0即用户未给此电影评分。
第三张是预测后的分数的图示，可以看到第三张图片的颜色与第一张相似（不包括第一张原来为蓝色的区域），这证明程序实现几乎没有问题（虽然这样的判断实际上并不严谨）。第二张图片表明Gradient Descent确实是趋于收敛的状态。

learning rate, regularization parameters等参数可以在主程序中调节，详见注释。
